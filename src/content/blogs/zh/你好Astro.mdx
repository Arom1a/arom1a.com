丢掉上一个写得臃肿至极还丑陋至极的主页后，用 [Astro](https://astro.build/) 又重新写了一个主页！

（其实上一个花了我很多心血折腾排版和功能 \>\_\<，
就当作是 Tailwind 初探索和练习吧，也算这一版的雏形。
毕竟“熟能生巧”嘛。。。）

安定下来后，这里就会是我位于飘忽不定的量子之海的小屋一栋啦。正如左边导航栏（或者手机端的上边）所展示，这个站点主要存放发布我写过的文章、我拍过的照片和翻唱过的歌。
文章将主要关于

- 尤其关于数学和计算机的笔记
- 有趣的小知识
- 缺少灵感的文学
- 长篇大论的想法
- 个人偏颇的评价

之前写过的东西应该也会被搬过来，一旦我有时间打磨打磨整理整理那些东西。。。

虽然本站的好多东西还没有完善（也有可能永远也不会被完善了 lol），但是已经可以当作比较精细的发布版本了（一些 WIP 的目标放在 README 里面了）。
如果你觉得任何元素有改进的空间，或者对本站有任何建议的话，请直接私信骚扰我，或者到 [repo](https://github.com/Arom1a/arom1a.com) 里直接提 issue 或者 pull request！！！
